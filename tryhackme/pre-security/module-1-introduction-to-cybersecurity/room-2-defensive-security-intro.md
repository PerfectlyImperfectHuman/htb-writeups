## What this room covers

A brief introduction to Defensive Security ( opposite of offensive security, here we do everything we can to prevent cyber attacks/intrusions from occuring and dealing with them in a proper way if they do occur) They are known as the Blue Team

## Key concepts learned

Some of the tasks that are related to defensive security include:

- User cyber security awareness — Basically securing the user by telling them about all the common attack patterns that hackers use to infiltrate their systems (like anonymous links etc)
- Documenting and managing assets — Managing all the devices connected to the network
- Updating and patching systems — if you find any exploit patch it until there is no way left for a hacker to enter
- Setting up preventative security devices — Setting up firewall and other such security measures to minimize the chances of an attack
- Setting up logging and monitoring devices — Properly logging everything so that everything is recorded

- Security Operations Center (SOC) — It is a team of professionals that monitor, prevent, detect, investigate respond to any kind of threats to the company etc
- Threat Intelligence — collects information to help the company better prepare against potential adversaries.
- Digital Forensics and Incident Response (DFIR) — It keeps investigating all the parts of the system to make sure there is no threat and in case of a threat/incident it responds properly
- Malware Analysis — Monitors, Prevents, Detect and Respond to any kind of virus attack (i.e trojan, ransomware etc) it is done by running the malware in a controlled environment and monitoring its activities
