## Key concepts learned

- Networking — two or more thing connected in one way or another for any reason form a network and in computer terms a network is formed when two or more devices are connected to one another
- Internet — Basically a nest of networks ( that have nested networks within them as well ) interconnected to one another forming a spider web like structure
- Internet Protocol (IP) Address — is a combination of 4 octet pairs and is used to identify a device on a network ( although it can be changed easily and faked as well ) and it has two types IPv4 ( which provides less addresses than the amount of devices that are connected to the Internet right now ) and so to solve this problem IPv6 is introduced which is a longer and a little bit more complicated version ( but it can make up to 370 Trillion addresses ) so the problem is solved
- Media Access Control (MAC) Address — It is a type of address which is unique for each and every device on the network ( like a fingerprint ) although it can also be changed using a method called "Spoofing" it is like a serial number for a device ( found on the motherboard and added the computer production company )
- Public Network — It is basically a network that is freely connected to the internet and can be accessed easily (like at work)
- Private Network — It is the type of network that we use at home and each device connected has a different private IP but the same Public IP
- PING — it basically used the ICMP to send and receive packets and tells us about the status of a IP Address and it is a very important and fundamental tool in networking
